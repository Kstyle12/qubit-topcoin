Why QTOP

Built for a world
after quantum computing.

CPU-native mining

RandomX is engineered to run best on ordinary CPUs. No ASICs. No GPU farms. A laptop mines on equal terms with a datacenter. Forever.

Get started

Mine in four steps.
No hardware required.

If your computer has a CPU — and it does — you are a first-class participant in the QTOP network.

01
Clone and install
One command on Mac or Linux.
git clone https://github.com/Kstyle12/qubit-topcoin.git
cd qubit-topcoin
chmod +x install.sh && ./install.sh
02
Create a wallet
FALCON-512 keypair, AES-256-GCM encrypted.
cd qtopcore

cargo run --bin qtopwallet -- create
03
Start a node
Connects to peers automatically.
cargo run --bin qtopcore -- 5003 YOUR_QTOP_ADDRESS
04
Mine QTOP
Rewards go straight to your wallet.
cargo run --bin miner -- YOUR_QTOP_ADDRESS

FAQ
Honest answers
to fair questions.

These are the questions people actually ask. We answer them directly.

Is QTOP real? Is this a scam?
QTOP is a real open source cryptocurrency project. Every line of code is public on GitHub. There is no premine, no ICO, no investment ask, and no way to buy QTOP — the only way to get it is to mine it. Judge it by the code, not by trust.
When is mainnet launching?
Mainnet is targeted for Q3 2027. QTOP will not launch until the software has been independently security audited and the testnet thoroughly stress tested. The Cori Testnet is active now. Testnet coins have no monetary value.
How is QTOP different from Bitcoin?
Bitcoin uses ECDSA signatures which are vulnerable to quantum computers. QTOP uses FALCON-512 — standardized by NIST in 2024 — resistant to all known quantum algorithms. QTOP also uses RandomX for mining, designed to run best on ordinary CPUs so anyone can participate equally.
Why should I trust something built by a high schooler?
You should not trust it because of who built it — evaluate it by what was built. Every line of code is public. The cryptographic primitives are NIST-standardized algorithms, not invented cryptography. A security audit is planned before mainnet. Age is irrelevant — the code either works or it does not, and you can verify that yourself.
Why should I trust a coin built with AI assistance?
The same reason you would trust any open source project — because you can read the code. AI was used as a coding partner and teacher, not a black box. Every decision was understood and deliberately made. The cryptographic algorithms are NIST standards. If anything is wrong you can find it — and we want you to.
Can I make money mining on testnet?
No. Testnet coins have no monetary value and will not transfer to mainnet. Mining on the Cori Testnet is purely for testing the network. There is no promise of future value and no investment is being solicited.
Will my Mac be damaged by mining?
No. RandomX runs your CPU at high utilization — similar to rendering a video. Modern CPUs have built-in thermal protection. Your Mac will get warm and fans will spin up. You can stop at any time by closing the terminal.
What happens to my testnet coins when mainnet launches?
Testnet coins do not carry over to mainnet. This is standard practice across the entire cryptocurrency industry. When mainnet launches everyone starts from zero on equal terms. No premine. No advantage for early testers. No advantage for the developer.
